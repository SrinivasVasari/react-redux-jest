## Project Starts with

In the project directory, you can run:

1. `git clone https://github.com/SrinivasVasari/react-redux-jest.git`,
2. `cd react-redux-jest`
3. `npm install`
4. `npm start`
5. `open in browser http://localhost:3000/`
## For test cases

1. npm test
